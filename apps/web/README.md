# FUTSTATS Web

Aplicação web/PC do FUTSTATS.

Esta área deve concentrar experiências que funcionam melhor em telas maiores:

- gestão administrativa;
- revisão de partidas;
- relatórios;
- configurações avançadas;
- telas operacionais que não dependem do contexto de quadra.

A stack final ainda pode ser definida. A documentação atual considera web/PC como parte do produto multiplataforma, sem competir com o foco principal em mobile.
