# FUTSTATS Mobile

Aplicação principal do FUTSTATS para uso em celular e tablet.

Esta área deve concentrar a experiência de campo/quadra:

- entrada no app;
- criação e entrada em times;
- agenda e confirmação de presença;
- escalação;
- operação ao vivo da partida;
- perfil do atleta;
- fluxos sociais ligados ao uso em tempo real.

Stack prevista:

- Expo;
- React Native;
- Supabase client;
- componentes compartilhados vindos de `packages/shared`.
