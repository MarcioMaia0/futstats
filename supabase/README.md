# FUTSTATS Supabase

Estrutura de banco, autenticação, storage, policies e funções ligadas ao Supabase.

Esta área deve concentrar:

- migrations;
- seeds;
- policies de RLS;
- configurações de storage;
- funções SQL;
- documentação operacional específica do banco.

Enquanto o projeto Supabase estiver instável ou com erro no painel, não aplicar migrations diretamente.
