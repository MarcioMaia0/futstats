# Design System Integration

## Objetivo

Definir como o frontend aplica identidade visual.

## Temas

Cada tema deve expor tokens:

- `color.primary`
- `color.secondary`
- `color.accent`
- `color.background`
- `color.surface`
- `color.text`
- `color.danger`
- `color.success`

## Regra

Componentes não devem usar cores fixas quando o valor puder vir do tema.
