# User Journeys

## Primeiro acesso autenticado

1. Entrar ou criar conta.
2. Completar perfil mínimo quando necessário.
3. Escolher um caminho inicial:
   - criar meu time;
   - entrar em um time;
   - explorar primeiro.

## Jornada casual

1. Entrar ou explorar.
2. Acompanhar times e conteúdo.
3. Buscar time ou jogador.
4. Ver resultado.
5. Reagir, comentar ou seguir quando fizer sentido.

## Jornada do gestor

1. Criar time.
2. Configurar identidade básica do time.
3. Adicionar elenco.
4. Agendar ou registrar jogo.
5. Acompanhar histórico e gestão.

## Jornada do atleta

1. Entrar em um time.
2. Enviar solicitação.
3. Aguardar aprovação enquanto explora o app.
4. Completar dados esportivos quando aprovado e quando necessário.
5. Acompanhar jogos, time e perfil esportivo.

## Jornada analítica

1. Registrar elenco.
2. Registrar escalação.
3. Lançar eventos.
4. Consolidar estatísticas.
5. Analisar padrões.
6. Tomar decisões.

## Jornada social

1. Seguir time.
2. Ver resultado.
3. Curtir.
4. Comentar.
5. Compartilhar.
