# User Journeys

## Jornada casual

1. Criar ou entrar em um time.
2. Registrar jogo.
3. Marcar placar.
4. Adicionar autores dos gols.
5. Compartilhar card.
6. Ver resenha.

## Jornada do diretor

1. Criar time.
2. Adicionar elenco.
3. Agendar jogo.
4. Registrar adversário.
5. Finalizar partida.
6. Acompanhar histórico.

## Jornada analítica

1. Registrar elenco.
2. Registrar escalação.
3. Lançar eventos.
4. Consolidar estatísticas.
5. Analisar padrões.
6. Tomar decisões.

## Jornada social

1. Seguir time.
2. Ver resultado.
3. Curtir.
4. Comentar.
5. Compartilhar.
