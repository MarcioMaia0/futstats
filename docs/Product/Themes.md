# Themes

## Objetivo

Permitir que times e usuários personalizem a experiência visual.

## Regras

1. Máximo de três cores principais.
2. Tema do time e tema do usuário são separados.
3. Modo claro/escuro é independente das cores.
4. Cards compartilháveis devem usar identidade visual do time.
5. Acessibilidade deve prevalecer sobre estética.

## Cores sugeridas

- Primary color.
- Secondary color.
- Accent color.
