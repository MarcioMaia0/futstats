# Backup and Restore


## Objetivo

Definir estratégia de backup.

## Regras

1. Backup diário do banco.
2. Backup de mídia separado.
3. Teste periódico de restore.
4. Retenção definida por ambiente.
5. Logs de restore devem ser documentados.

## Prioridade

Histórico de partidas, gols e vínculos de jogadores são dados críticos.
