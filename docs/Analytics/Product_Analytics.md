# Product Analytics

## Objetivo

Definir eventos de analytics.

## Eventos iniciais

- `account_created`
- `team_created`
- `match_created`
- `goal_registered`
- `match_card_shared`
- `player_tagged`
- `comment_created`
- `statistics_viewed`
- `referee_review_created`

## Métricas principais

- Ativação.
- Retenção.
- Compartilhamento.
- Profundidade.
- Conversão para recursos avançados.
