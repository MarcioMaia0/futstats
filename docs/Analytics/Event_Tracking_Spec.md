# Event Tracking Spec

## Eventos essenciais

- `account_created`
- `team_created`
- `match_created`
- `goal_added`
- `match_finished`
- `match_card_generated`
- `match_card_shared`
- `player_profile_viewed`
- `team_profile_viewed`
- `post_reacted`

## Métrica norte

Número de partidas registradas e compartilhadas por semana.

## Métricas secundárias

- Taxa de criação de time.
- Taxa de finalização de partida.
- Taxa de compartilhamento.
- Retenção semanal.
- Partidas por time.
