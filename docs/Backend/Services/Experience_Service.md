# Experience Service

## Objetivo

Temas, linguagem e preferências.

## Responsabilidades

- Validar regras do domínio.
- Executar casos de uso relacionados.
- Integrar repositórios.
- Emitir eventos quando necessário.

## Não responsabilidades

- Renderizar interface.
- Definir textos de UI.
- Acessar banco diretamente fora dos repositórios.
