# Card Generation Service


## Objetivo

Definir serviço de geração de cards.

## Cards iniciais

- resultado do jogo;
- artilheiro;
- jogador destaque;
- próximo jogo;
- ranking semanal;
- card de atleta.

## Regras

1. Cards devem respeitar tema do time.
2. Cards devem ter fallback visual.
3. Cards devem ser compartilháveis.
4. Geração pode ser assíncrona.
5. Conteúdo deve respeitar privacidade.
