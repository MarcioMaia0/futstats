# ADR 002: Progressive Product Complexity

## Status

Draft

## Contexto

O FUTSTATS atende usuário casual e analítico.

## Decisão

Complexidade será progressiva. Recursos avançados não bloqueiam uso básico.

## Consequências

- Melhor adoção inicial.
- UX mais simples.
- Arquitetura preparada para profundidade.
