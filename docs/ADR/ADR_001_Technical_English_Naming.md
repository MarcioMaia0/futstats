# ADR 001: Technical English Naming

## Status

Draft

## Contexto

O projeto terá banco, APIs, código, enums e documentação técnica.

## Decisão

Toda nomenclatura técnica será em inglês.

## Consequências

- Facilita manutenção.
- Facilita uso de bibliotecas.
- Facilita trabalho com IA.
- Evita mistura entre regra de negócio e texto de interface.
