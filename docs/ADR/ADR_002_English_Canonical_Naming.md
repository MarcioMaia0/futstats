# ADR 002: English Canonical Naming

## Contexto

O sistema terá UI em português e modos de linguagem, mas código e banco precisam ser consistentes.

## Decisão

Usar inglês técnico em tabelas, colunas, APIs, enums, funções e variáveis.

## Consequências

Facilita manutenção, IA, integração e internacionalização.
