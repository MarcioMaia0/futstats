# Separação entre user e player

## Status

Draft

## Contexto

A documentação e arquitetura do FUTSTATS precisam registrar decisões para evitar retrabalho futuro.

## Decisão

Usuário da plataforma e perfil esportivo serão entidades separadas.

## Consequências

- Facilita manutenção.
- Reduz ambiguidade.
- Ajuda pessoas e IA a entenderem o projeto.
- Exige disciplina documental.
