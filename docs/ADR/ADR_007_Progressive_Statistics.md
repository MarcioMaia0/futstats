# ADR 007 Progressive Statistics


## Status

Draft

## Contexto

Estatísticas avançadas são valiosas, mas podem afastar usuários casuais.

## Decisão

Métricas serão progressivas e dependentes dos dados disponíveis.

## Consequências

O produto atende casual e analítico sem criar barreira de entrada.
