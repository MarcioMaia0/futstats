# Viral Loops

## Objetivo

Definir os mecanismos de crescimento viral do FUTSTATS.

## Loop 1: Resultado compartilhado

1. Jogo é registrado.
2. Card do resultado é gerado.
3. Card é compartilhado.
4. Jogadores e amigos acessam.
5. Novos usuários seguem time ou criam conta.

## Loop 2: Gol marcado

1. Gol é associado a atleta.
2. Atleta recebe destaque.
3. Atleta compartilha perfil ou card.
4. Amigos acessam.
5. Novos atletas querem ser marcados também.

## Loop 3: Ranking

1. Time registra jogos.
2. Ranking de artilharia aparece.
3. Jogadores disputam posição.
4. Mais jogos são registrados.
5. Ranking fica mais valioso.

## Loop 4: Resenha

1. Resultado gera conversa.
2. Comentários e reações aumentam engajamento.
3. Usuários voltam para acompanhar.
4. Time mantém hábito de uso.

## Restrições

- Viralidade não pode depender de conteúdo ofensivo.
- Corneta deve ter controle e moderação.
- Compartilhamento deve respeitar privacidade do time e do atleta.
