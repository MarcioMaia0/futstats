# Product Decisions

## PD-001: Usuário casual é tão importante quanto usuário analítico

O FUTSTATS deve atender quem quer apenas registrar jogo e resenha, sem forçar scout.

## PD-002: Adoção inicial pelo social

O produto deve entrar pelo compartilhamento, placar, gols, perfil e resenha.

## PD-003: Inteligência é progressiva

Estatísticas avançadas são uma camada de maturidade, não requisito inicial.

## PD-004: Legado é consequência

Memória histórica é valiosa, mas não é o principal argumento inicial de venda.

## PD-005: Recursos avançados opcionais

Todo recurso profundo deve agregar sem bloquear o uso simples.
