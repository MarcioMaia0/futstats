# PRD: Advanced Statistics

## Objetivo

Definir a camada avançada de estatísticas do FUTSTATS para times que desejam estudar performance.

## Princípio

Estatísticas avançadas são opcionais e progressivas.

## Funcionalidades

- Plus/Minus.
- Minutagem.
- Escalações.
- Substituições.
- Scout de chutes.
- Scout de passes.
- Scout defensivo.
- Performance por quadra.
- Performance por adversário.
- Relatório pós-jogo.

## Pré-condições

- Para Plus/Minus: escalação inicial e substituições.
- Para análise de quadra: local da partida cadastrado.
- Para análise avançada: eventos suficientes.

## UX

A camada avançada deve aparecer como "modo análise" ou "lançar scout", nunca como requisito para registrar jogo.

## Métricas

- Adoção de scout.
- Partidas com escalação.
- Partidas com eventos avançados.
- Relatórios gerados.
