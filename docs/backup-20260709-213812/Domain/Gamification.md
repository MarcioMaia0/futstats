# Gamification Domain

## Objetivo

Definir como elementos de jogo, conquistas, cards, rankings e reputação serão usados para engajar sem distorcer comportamento.

## Princípios

1. Gamificação deve aumentar diversão.
2. Não deve incentivar toxicidade.
3. Não deve punir usuários com poucos dados.
4. Deve separar resenha de métrica séria.
5. Deve ser visual e compartilhável.

## Elementos

- Player Card.
- Overall.
- Badges.
- Conquistas.
- Rankings.
- Destaques da rodada.
- Artilheiro.
- Rei da assistência.
- Mural de resenha.

## Regras

1. Overall deve explicar sua base de cálculo.
2. Rankings devem ter filtros.
3. Conquistas devem ser auditáveis quando dependerem de estatísticas.
4. Modo resenha pode usar termos divertidos.
5. Modo técnico deve manter credibilidade.

## Decisão

Gamificação é ponte entre social e estatística.
