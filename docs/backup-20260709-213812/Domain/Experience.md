# Experience Domain

## Objetivo

Definir temas, linguagem e personalização.

## Regras

1. Usuário pode escolher tema pessoal.
2. Time pode definir tema padrão.
3. Máximo de três cores.
4. Linguagem muda interface, não dado.
5. Dados canônicos ficam em inglês.
6. UI traduz conforme modo.

## Modos

- `TECHNICAL`
- `VARZEA`
- `RESENHA`
