# Social Sharing Flow


## Objetivo

Definir fluxo de compartilhamento social.

## Fluxo

1. Usuário finaliza partida.
2. App gera card.
3. Usuário escolhe canal.
4. Compartilha.
5. Pessoas acessam link ou imagem.
6. Plataforma estimula cadastro ou acompanhamento.

## Regras

1. Compartilhamento deve ser rápido.
2. Card deve ser visualmente forte.
3. Link deve respeitar privacidade.
4. Fluxo não deve exigir scout.
