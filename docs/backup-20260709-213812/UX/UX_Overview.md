# UX Overview

## Princípio

A experiência do FUTSTATS deve ser simples no início e profunda quando necessário.

## Regras

- Mobile first.
- Poucos campos no primeiro uso.
- Modo rápido evidente.
- Modo avançado acessível, mas não obrigatório.
- Compartilhamento como ação natural.
- Resenha como parte legítima do produto.
