# Advanced Scout Flow

## Objetivo

Definir fluxo avançado de análise.

## Fluxo

1. Escolher partida.
2. Confirmar elenco.
3. Definir escalação.
4. Registrar eventos.
5. Validar timeline.
6. Finalizar scout.
7. Gerar relatório.

## Regra

Este fluxo não deve interferir no uso casual.
