# Routing

## Rotas principais

- `/`
- `/login`
- `/teams`
- `/teams/:teamId`
- `/teams/:teamId/matches`
- `/matches/:matchId`
- `/players/:playerId`
- `/feed`
- `/settings`
- `/profile`

## Regra

Rotas devem refletir domínios, não apenas telas momentâneas.
