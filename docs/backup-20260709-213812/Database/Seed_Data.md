# Seed Data

## Objetivo

Definir dados iniciais necessários para ambiente local, testes e demonstrações.

## Dados recomendados

- roles padrão;
- modos de linguagem;
- vocabulário inicial;
- temas básicos;
- tipos de partida;
- tipos de piso;
- status de match;
- critérios de arbitragem;
- métricas padrão.

## Regra

Seeds devem ser idempotentes.
