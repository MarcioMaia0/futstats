# Data Privacy

## Objetivo

Definir cuidados de privacidade.

## Dados sensíveis

- e-mail;
- telefone;
- foto;
- localização;
- dados de menores de idade;
- comentários e avaliações;
- histórico de presença.

## Regras

1. Perfis públicos devem ter controle de visibilidade.
2. Dados de contato não devem ser públicos por padrão.
3. Jogadores devem poder controlar informações exibidas.
4. Dados de menores exigem política específica.
5. Dados anonimizados podem ser usados para analytics e estudos.
6. Exclusão de conta não deve quebrar histórico de partidas.

## Estratégia

Separar dados pessoais de dados esportivos sempre que possível.
