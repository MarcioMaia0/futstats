# MVP Sprint Plan

## Sprint 1: Foundation

- Auth básico.
- Users.
- Teams.
- Design tokens.
- Estrutura de navegação.

## Sprint 2: Quick Match

- Criar partida.
- Registrar placar.
- Registrar gols.
- Histórico simples.

## Sprint 3: Sharing

- Card de resultado.
- Compartilhamento.
- Perfil básico de time.
- Perfil básico de jogador.

## Sprint 4: Social Basic

- Feed do time.
- Post automático de partida.
- Reações simples.

## Sprint 5: Statistics Basic

- Artilharia.
- Histórico de resultados.
- Estatísticas simples.

## Sprint 6: Polish and Launch

- QA.
- Observabilidade.
- Privacidade.
- Ajustes de UX.
- Beta fechado.
