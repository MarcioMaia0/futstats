# ADR 010: MVP Casual First

## Status

Accepted

## Contexto

O FUTSTATS pode crescer para estatísticas avançadas, mas a adoção inicial tende a vir pelo uso casual e social.

## Decisão

O MVP será casual-first, focando criação rápida de partida, placar, gols, card e compartilhamento.

## Consequências

Positivas:

- Menor fricção.
- Maior chance de viralização.
- Teste de mercado mais rápido.

Negativas:

- Estatísticas avançadas ficam para fases posteriores.
- Times analíticos podem precisar esperar recursos profundos.
