# AI Insight Generation


## Objetivo

Definir uso futuro de IA para gerar insights.

## Casos

- resumo pós-jogo;
- padrões de desempenho;
- alertas de adversário;
- sugestão de escalação;
- análise de queda de performance;
- explicação simples de métricas.

## Regras

1. IA não deve inventar dados.
2. IA deve indicar quando dados são insuficientes.
3. IA deve respeitar privacidade.
4. IA deve diferenciar insight de recomendação.
