# AI Strategy

## Objetivo

Definir como IA pode ser usada no FUTSTATS.

## Princípio

IA deve gerar insights, não substituir a decisão humana.

## Casos futuros

- Resumo pós-jogo.
- Sugestões táticas.
- Análise de padrões.
- Detecção de inconsistências.
- Geração de texto para resenha.
- Moderação de conteúdo.
- Recomendações para técnicos.

## Cuidados

- Não inventar dados.
- Explicar incerteza.
- Respeitar privacidade.
- Separar insight de fato.
