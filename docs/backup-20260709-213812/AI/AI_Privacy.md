# AI Privacy

## Regras

1. Não enviar dados sensíveis sem consentimento.
2. Permitir anonimização.
3. Explicar uso de IA.
4. Diferenciar conteúdo gerado de conteúdo humano.
5. Não usar avaliações privadas em conteúdo público.
