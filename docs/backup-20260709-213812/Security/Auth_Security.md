# Auth Security

## Regras

1. Senhas nunca armazenadas em texto puro.
2. Preferir OAuth quando possível.
3. Tokens com expiração.
4. Refresh tokens protegidos.
5. Rate limit em login.
6. Auditoria em mudanças sensíveis.
