# Security Principles

## Princípios

- Autorização por contexto.
- Proteção de dados pessoais.
- Histórico preservado com auditoria.
- Denúncias e moderação.
- Mínimo privilégio.
- Validação server-side.

## Áreas sensíveis

- Permissões de time.
- Perfis de menores.
- Conteúdo social.
- Reivindicação de atleta.
- Dados de localização.
