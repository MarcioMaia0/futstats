# UX Principles

## Princípios

1. O primeiro uso deve ser simples.
2. O modo rápido vem antes do modo avançado.
3. O usuário casual não deve se sentir errado por não preencher dados.
4. A resenha deve ser natural.
5. A profundidade deve aparecer por convite, não por obrigação.
6. Compartilhar deve ser fácil.
7. A interface deve falar a linguagem escolhida pelo usuário.
