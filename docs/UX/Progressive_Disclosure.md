# Progressive Disclosure


## Objetivo

Definir progressividade na interface.

## Regra central

Mostrar primeiro o mínimo necessário. Revelar profundidade quando o usuário demonstrar intenção.

## Exemplos

- Criar partida simples antes de partida avançada.
- Registrar gol antes de pedir assistência.
- Mostrar ranking simples antes de relatórios complexos.
- Permitir scout somente para quem ativar modo avançado.

## Benefício

Reduz fricção para casual sem remover profundidade para analítico.
