# Match Flow

## Modo rápido

1. Criar partida.
2. Informar adversário.
3. Registrar placar.
4. Marcar gols.
5. Compartilhar.

## Modo avançado

1. Agendar.
2. Selecionar elenco.
3. Definir escalação.
4. Registrar eventos.
5. Finalizar.
6. Gerar relatório.

## Regra

Modo rápido deve ser padrão.
