# Advanced Match Flow

## Objetivo

Permitir registro profundo para times organizados.

## Fluxo

1. Agendar partida.
2. Definir adversário.
3. Definir quadra.
4. Fazer check-in.
5. Definir titulares.
6. Registrar eventos.
7. Finalizar.
8. Gerar relatório.

## Regra

Deve ser acessível como opção avançada, não fluxo obrigatório.
