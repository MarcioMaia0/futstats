# Advanced Scout Flow

## Objetivo

Fluxo opcional de scout.

## Etapas

1. Entrada do usuário.
2. Validação simples.
3. Confirmação.
4. Feedback visual.
5. Próxima ação recomendada.

## Cuidados

Este fluxo não deve adicionar fricção desnecessária ao uso casual.
