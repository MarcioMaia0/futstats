# Onboarding Flow

## Objetivo

Criar entrada simples para diferentes perfis.

## Fluxo casual

1. Entrar.
2. Criar ou entrar em time.
3. Registrar partida.
4. Compartilhar resultado.

## Fluxo diretor

1. Criar time.
2. Configurar identidade.
3. Adicionar elenco.
4. Criar partida.

## Regra

Onboarding não deve obrigar o usuário a escolher tudo no primeiro acesso.
