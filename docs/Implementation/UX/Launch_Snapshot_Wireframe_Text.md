# Launch Snapshot Wireframe Text

## Objetivo

Registrar um wireframe textual do recorte inicial de lançamento.

## Tela: Criar Partida Rápida

Campos:

- Adversário
- Data
- Placar do seu time
- Placar do adversário
- Botão: Adicionar gols
- Botão: Gerar card

## Tela: Adicionar Gol

Campos:

- Jogador
- Minuto opcional
- Assistência opcional

## Tela: Resultado

Elementos:

- Escudo do time
- Nome do adversário
- Placar
- Artilheiros
- Botão compartilhar

## Tela: Perfil do Jogador

Elementos:

- Foto
- Apelido
- Gols
- Jogos
- Últimos jogos
