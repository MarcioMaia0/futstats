# Launch Snapshot Privacy Rules

## Regras

1. Perfil de usuário deve ter controles de privacidade.
2. Perfil de jogador pode ser público, privado ou restrito ao time.
3. Partidas podem ser públicas ou privadas.
4. Comentários podem ser denunciados.
5. Usuário pode remover foto.
6. Dados históricos de partida não devem ser apagados sem política clara.

## Recorte inicial

Começar com privacidade simples:

- Time público/privado.
- Jogador público/privado.
- Posts denunciáveis.
