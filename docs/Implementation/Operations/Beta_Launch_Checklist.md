# Beta Launch Checklist

## Produto

- Fluxo de partida rápida validado.
- Card compartilhável funcionando.
- Perfil de time funcionando.
- Perfil de jogador funcionando.

## Técnico

- Backup configurado.
- Logs mínimos.
- Monitoramento de erros.
- Ambiente de staging.
- Ambiente de produção.

## Segurança

- Auth testado.
- Permissões básicas.
- Denúncia de conteúdo.

## Analytics

- Eventos principais configurados.
- Métrica norte acompanhada.
- Funil de criação de partida medido.
