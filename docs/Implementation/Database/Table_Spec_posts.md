# Table Spec posts

## Objetivo

Especificar tabela `posts`.

## Finalidade

Representar conteúdo social.

## Campos sugeridos

- `id`
- `author_user_id`
- `team_id`
- `match_id`
- `post_type`
- `content`
- `visibility`
- `created_at`

## Regras

- Visibilidade deve ser respeitada.
- Conteúdo pode ser denunciado.
