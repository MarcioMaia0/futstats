# Acceptance Criteria

## Padrão

Toda feature deve conter:

- objetivo;
- usuário-alvo;
- pré-condições;
- fluxo principal;
- fluxo alternativo;
- critérios de aceite;
- eventos gerados;
- impacto em métricas.
