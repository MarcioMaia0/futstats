# Domain Services

## Serviços principais

| Serviço | Responsabilidade |
|---|---|
| IdentityService | Contas, usuários e papéis |
| TeamService | Times e configurações |
| PlayerService | Perfis esportivos |
| MatchService | Partidas |
| ScoreService | Placar e gols |
| StatisticsService | Métricas |
| SocialService | Feed e interações |
| RefereeService | Arbitragem |
| ExperienceService | Temas e linguagem |
| MediaService | Arquivos e cards |

## Regra

Serviços devem refletir linguagem de domínio, não telas.
