# C4 Containers

## Containers principais

### Mobile App

Cliente principal para uso casual, registro rápido de jogos, placar, gols, feed, perfil e resenha.

### Web App

Cliente administrativo e analítico para diretoria, relatórios, scouts, gestão de elenco e dados mais profundos.

### API Backend

Camada de aplicação responsável por autenticação, permissões, orquestração de domínios, APIs públicas e integração com serviços externos.

### Database

Persistência relacional dos dados canônicos da plataforma.

### Media Storage

Armazenamento de escudos, fotos, vídeos, thumbnails, cards gerados e assets visuais.

### Analytics Store

Camada futura para dados agregados, métricas, eventos e inteligência.

### AI Services

Camada futura para insights, recomendações, sumarização e análise assistida.
