# Product Metrics

## Métricas principais

- partidas criadas;
- partidas finalizadas;
- gols registrados;
- cards compartilhados;
- usuários ativos;
- times ativos;
- uso de recursos sociais;
- uso de recursos avançados;
- retenção por time;
- conversão para recursos pagos.

## Métrica norte

Times ativos com partidas registradas e compartilhadas.
