# North Star Metric


## Objetivo

Definir métrica norte.

## Proposta

Número de partidas registradas e compartilhadas por semana.

## Justificativa

Essa métrica combina uso real, valor social e criação de dados para evolução futura.

## Métricas auxiliares

- times ativos;
- jogadores marcados em gols;
- cards compartilhados;
- jogos finalizados;
- usuários recorrentes.
