# Match Flow

## Modo rapido

1. Criar partida.
2. Informar adversario.
3. Registrar placar.
4. Marcar gols por toque no escudo do time ou do adversario.
5. Compartilhar.

## Modo avancado

1. Agendar.
2. Selecionar elenco do quadro.
3. Definir titulares em superficie visual de quadra ou campo.
4. Operar eventos ao vivo na mesma superficie.
5. Finalizar.
6. Revisar por video, se necessario.
7. Gerar relatorio.

## Regra

Modo rapido deve ser padrao.
