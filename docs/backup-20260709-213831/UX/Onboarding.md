# Onboarding

## Objetivo

Permitir entrada rápida.

## Fluxo casual

1. Entrar.
2. Criar ou procurar time.
3. Registrar jogo.
4. Compartilhar resultado.

## Fluxo diretor

1. Criar time.
2. Configurar identidade.
3. Adicionar elenco depois.
4. Registrar partida.

## Regra

Não pedir tudo no primeiro acesso.
