# Quick Match Flow

## Objetivo

Permitir registrar jogo rapidamente.

## Fluxo

1. Escolher/criar time.
2. Criar partida rápida.
3. Informar adversário.
4. Informar placar.
5. Adicionar autores dos gols opcionalmente.
6. Gerar card.
7. Compartilhar.

## Regras

- Não exigir elenco completo.
- Não exigir quadra.
- Não exigir scout.
- Não exigir árbitro.
