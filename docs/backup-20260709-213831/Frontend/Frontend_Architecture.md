# Frontend Architecture

## Objetivo

Definir arquitetura frontend do FUTSTATS.

## Princípios

- Mobile first.
- Design system único.
- Componentes reutilizáveis.
- Estado global mínimo.
- Feature flags para recursos avançados.
- Separação entre dado canônico e texto exibido.

## Camadas

- Screens
- Components
- Hooks/Composables
- Services
- State
- Theme
- i18n
