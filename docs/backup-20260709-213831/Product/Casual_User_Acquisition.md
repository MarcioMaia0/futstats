# Casual User Acquisition

## Objetivo

Documentar como o FUTSTATS deve conquistar usuários casuais.

## Premissa

O usuário casual não quer preencher cadastro longo. Ele quer resolver algo imediato: registrar o jogo, mostrar o placar, marcar quem fez gol e compartilhar.

## Gatilhos de aquisição

- Card bonito do resultado.
- Link de convite para atletas do jogo.
- Perfil do jogador com gols.
- Ranking simples de artilharia.
- Resenha e comentários.
- Compartilhamento no WhatsApp e Instagram.

## Fluxo ideal

1. Diretor ou atleta registra partida simples.
2. Sistema gera card compartilhável.
3. Jogadores marcados recebem ou acessam link.
4. Atletas entram para ver gols, perfil e resenha.
5. Alguns passam a seguir o time.
6. O time cria hábito de registrar jogos.

## Critérios de sucesso

- Criar partida em menos de 60 segundos.
- Compartilhar resultado em até 2 toques após finalizar.
- Permitir marcar jogador mesmo com perfil mínimo.
- Não exigir scout, escalação ou quadra no primeiro uso.

## Risco

Se a primeira experiência parecer trabalho administrativo, o usuário casual abandona antes de entender o valor.
