# UX Philosophy

## Princípio

A UX deve ser progressiva.

O primeiro uso precisa ser simples. A profundidade aparece conforme o usuário demonstra interesse.

## Regras

- Mobile first.
- Menos campos no início.
- Modo rápido antes de modo avançado.
- Compartilhamento visível.
- Resenha tratada como parte da experiência.
- Dados avançados escondidos até fazerem sentido.
