# AI Data Boundaries

## Objetivo

Definir limites de dados para IA.

## Regras

1. Não expor dados privados sem permissão.
2. Não inventar estatísticas.
3. Não usar conteúdo privado para feed público.
4. Separar insights individuais e coletivos.
5. Permitir opt-out quando necessário.
