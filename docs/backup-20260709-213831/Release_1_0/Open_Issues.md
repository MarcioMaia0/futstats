# Open Issues

## Objetivo

Registrar decisões que ainda precisam ser detalhadas no futuro.

## Produto

- Definir branding final.
- Definir nomes comerciais dos planos.
- Definir limites do plano gratuito.
- Definir se haverá verificação de times no MVP.
- Definir política de conteúdo público.

## Técnico

- Escolher stack final.
- Definir banco final.
- Definir infraestrutura inicial.
- Definir estratégia de upload de mídia.
- Definir provedor de autenticação.
- Definir geração de imagem de cards.

## UX

- Criar wireframes visuais.
- Criar protótipo navegável.
- Definir design system visual.
- Testar linguagem Resenha com usuários reais.

## Dados

- Definir ERD final.
- Definir SQL inicial.
- Definir regras de recalculo de estatísticas.
- Definir critérios de confiabilidade das métricas.
