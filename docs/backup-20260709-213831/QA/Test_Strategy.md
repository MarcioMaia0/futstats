# Test Strategy

- Unitários.
- Integração.
- End-to-end.
- Contrato de API.
- Testes de permissão.
- Testes de regressão estatística.
- Testes de UX crítica.

- Criar partida rápida.
- Registrar gol.
- Editar gol.
- Finalizar partida.
- Recalcular estatísticas.
- Compartilhar card.
