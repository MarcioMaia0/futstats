# Data Consistency Tests


## Objetivo

Definir testes de consistência de dados.

## Casos

1. Gol cria placar correto.
2. Remover gol recalcula placar.
3. Jogador removido mantém histórico.
4. Partida cancelada não conta como finalizada.
5. Avaliação de árbitro pertence à partida.
6. Alterar linguagem não altera dado canônico.
