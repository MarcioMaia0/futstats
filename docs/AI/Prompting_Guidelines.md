# Prompting Guidelines

## Objetivo

Criar diretrizes para uso de IA com a documentação do FUTSTATS.

## Regras

1. Sempre informar o domínio relevante.
2. Pedir para respeitar Product Principles.
3. Pedir para não tornar recurso avançado obrigatório.
4. Pedir para separar dado técnico de UI.
5. Pedir para documentar trade-offs.

## Exemplo

"Leia Product_Principles.md, Matches.md e API_Conventions.md. Implemente criação rápida de partida sem exigir scout."
