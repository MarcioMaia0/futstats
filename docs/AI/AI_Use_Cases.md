# AI Use Cases

## Casos de uso

- Gerar resumo pós-jogo.
- Sugerir insights de performance.
- Explicar estatísticas.
- Detectar padrões.
- Ajudar técnico a interpretar dados.
- Moderar conteúdo.
- Sugerir textos de compartilhamento.
- Auxiliar suporte.

## Regra

Toda resposta de IA deve indicar quando depende de dados incompletos.
