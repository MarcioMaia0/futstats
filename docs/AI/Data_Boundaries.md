# AI Data Boundaries

## Regras

1. Não expor dados pessoais sem autorização.
2. Preferir dados agregados.
3. Explicar limitações de insights.
4. Não gerar acusações sobre atletas ou árbitros.
5. Manter rastreabilidade de dados usados.
