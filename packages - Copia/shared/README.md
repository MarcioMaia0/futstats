# FUTSTATS Shared

Pacote compartilhado entre mobile, web e backend.

Esta área deve guardar código reutilizável e contratos comuns, por exemplo:

- tipos de domínio;
- enums;
- validadores;
- DTOs;
- helpers puros;
- tokens de design;
- contratos de API;
- regras reutilizáveis que não dependem diretamente de UI ou banco.

Regra principal: código compartilhado não deve depender de detalhes específicos de Expo, navegador ou Supabase client.
